Zed Zed Zed update patch

This zip contains only the files changed for this update.
Replace the matching files in your GitHub repository with these versions, then run the GitHub build workflow again.

Changed in this patch:
- zombies use type labels instead of human names
- zombies strip all usable items and carried items
- zombies are slower and weaker
- zombies only alert and attack when they see living prey nearby
- zombies huddle and edge-shamble instead of beelining for colonists
- map zombie counts are capped by colonist count and time of day
- nightly and trickle spawning are toned down
