# Zed Zed Zed

Zed Zed Zed is a zombie mod for RimWorld built around steady pressure instead of one big raid at a time. The dead drift in from the edges, bunch up, push the base, claw out of the ground, spill out of graves, and sometimes cross the whole map in a herd.

The goal is simple. Zombies should be easy to kill, but dangerous in numbers. Noise should matter. Nights should feel worse. Each strain should have its own flavor without turning the mod into a mess.

## What it adds

- A full roster of zombie types, including biters, bone biters, runts, boomers, sick, drowned, brutes, grabbers, lurkers, and a few special variants.
- Ongoing map pressure through trickles, herd crossings, moon events, graves, and ground bursts.
- A staged infection system, bile med kits, and bile surgery that can turn a pawn into a colony lurker.
- Rotten flesh, rotten leather, zombie jerky, bile, gold teeth, and a small set of themed furniture and floors.
- A HUD, full settings menu, naming options, and debug tools.

## Current direction

- Zombies hit hard through pressure, not by being bullet sponges.
- Head and face hits are decisive.
- Drowned belong in water and get more dangerous in rain.
- Loud gunfire pulls more attention than quiet melee.
- Lurkers stay as the strange friendly exception.

## Build

This repository is the full source package for the mod. It includes the GitHub Actions workflow used to build the release zip.

This project was vibe coded with ChatGPT, then shaped by hand through testing, rewrites, and cleanup. The images and textures were generated first, then edited by hand to fit the mod.
